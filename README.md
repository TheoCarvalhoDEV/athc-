# Atchê - Plataforma de Eventos Cinematográfica 🎬

Bem-vindo ao **Atchê**, uma web-app de alta fidelidade voltada para a gestão e descoberta de eventos, festivais, clubes e atléticas. O Atchê não é apenas um site de ingressos; é a porta de entrada para a melhor experiência noturna.

Você é proprietário de uma casa noturna, organizador de festivais ou líder de atlética? O **Atchê** foi desenhado para elevar o nível da sua marca.

## ✨ Características do Projeto

O Atchê oferece uma experiência visual imersiva e de alta performance:

- **Estética "Velvet VIP"**: Interface moderna em modo escuro, com tons de dourado (#D4AF37) e tipografia refinada.
- **Feed de Fomo (Fear Of Missing Out)**: Layout focado em gerar urgência e desejo visual.
- **Área do Parceiro**: Sistema de login exclusivo para estabelecimentos gerenciarem seus eventos.
- **Animações Cinematográficas**: Transições fluidas usando **GSAP** (GreenSock Animation Platform) para micro-interações de luxo.
- **Busca e Filtros**: Localize eventos por nome ou filtre por categorias (Aberto, VIP, Hoje, Esta Semana).

## 🛠️ Tecnologias Utilizadas

- **Core**: [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Estilização**: [Tailwind CSS](https://tailwindcss.com/)
- **Animações**: [GSAP 3](https://greensock.com/gsap/) + [Lucide React](https://lucide.dev/) (Ícones)
- **Geolocalização**: [Google Maps API](https://developers.google.com/maps)
- **Roteamento**: [React Router Dom](https://reactrouter.com/)

## 🔧 Execução Local

Se você é desenvolvedor e deseja rodar o projeto localmente:

1. **Clonar o Repositório**:
   ```bash
   git clone https://github.com/TheoCarvalhoDEV/athc-.git
   ```

2. **Configurar Ambiente**:
   Crie um arquivo `.env` na raiz do projeto com sua chave de API:
   ```env
   VITE_GOOGLE_MAPS_API_KEY=SUA_CHAVE_AQUI
   ```

3. **Instalar e Rodar**:
   ```bash
   npm install
   npm run dev
   ```
